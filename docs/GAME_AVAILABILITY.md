# Pokémon Game Availability — Version Exclusives

Version-exclusive Pokémon per game pair, across all nine main-series generations. Only Pokémon that are unavailable in one or more versions of the same generation pair are listed — Pokémon available in all versions of a generation are omitted.

**Gen 1 database:** the `PokemonGameAvailability` table in `pokemon.db` is seeded with correct Gen 1 data (including `AvailabilityType`). Re-run `PokeApiConnector` to re-seed.
